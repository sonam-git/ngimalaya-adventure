Browser DevTools Check

How to verify:

  1. Right-click on any heading above
  2. Select "Inspect" or "Inspect Element"
  3. In the DevTools panel, look at the "Computed" tab
  4. Search for "font-family"
  5. It should show: "Jaini Purva", system-ui

Console Verification Script

Open browser console (F12) and run this:

const h2 = document.querySelector('h2');
console.log('Font:', window.getComputedStyle(h2).fontFamily);
// Should output: "Jaini Purva", system-ui
